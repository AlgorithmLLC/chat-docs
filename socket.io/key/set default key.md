### set default key(key, ack)
```javascript
key = {
  id: 15
}
```
Sets default user key.

Error types:
  - `wrong data provided`
  - `something went wrong`

Returns saved key.
```javascript
savedKey = {
  id: 15
  , user_id: 3
  , name: "NHKH75NB0/I4342wPK1Jon0rReb75UchVQ8vWqBjMzE="
  , is_default: true
  , created_at: "2014-08-19 14:28:10"
}
```
